// Algorithm Details

Place N queens on an N×N board such that no two queens attack each other (same row, column, or diagonal). Use backtracking: place a queen row by row, check constraints, and backtrack when invalid.
solve(row): if row==N → record solution
for col in 0..N: if isValid(row,col):
  place(row,col) → solve(row+1) → remove(row,col)
