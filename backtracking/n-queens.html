N-Queens Problem

Place N queens on an N×N chessboard so no two queens attack each other. Watch the backtracking recursion try every placement, detect conflicts, and backtrack in real time.

Backtracking Recursion Tree Constraint Check O(N!)
Problem Statement

Given an integer n, return all distinct solutions to the N-Queens puzzle. Each solution contains a distinct board configuration where 'Q' marks a queen and '.' marks an empty cell. No two queens can share the same row, column, or diagonal.

Backtracking Strategy
1
Place queens row by row — one queen per row is guaranteed, so iterate columns only.
2
isSafe check: scan vertically upward, upper-left diagonal, and upper-right diagonal. No need to check below (not yet placed).
3
If safe, place 'Q' and recurse to the next row.
4
Backtrack: after recursion returns, reset board[row][col] = '.' and try next column.
5
Base case: when row == n, all queens placed — snapshot the board and add to result.
isSafe in Detail
↑
Vertical: check board[i][col] for all i < row.
↖
Upper-left diagonal: walk (row−1,col−1), (row−2,col−2)…
↗
Upper-right diagonal: walk (row−1,col+1), (row−2,col+2)…
Complexity
TimeO(N!)
