Backtracking

Sudoku Solver

Watch backtracking fill a 9×9 Sudoku cell by cell — trying each digit 1–9, detecting row/column/box conflicts instantly, and unwinding when stuck. Every decision is logged live.

Backtracking Constraint Propagation O(9^m) Time 3 Puzzles
Problem Statement

Given a 9×9 partially filled Sudoku board, fill every empty cell (marked '.') such that: each row, each column, and each of the nine 3×3 sub-boxes contains all digits 1–9 exactly once. The puzzle is guaranteed to have exactly one solution.

Backtracking Strategy
1
Scan left-to-right, top-to-bottom for the first empty cell ('.').
2
Try placing digits 1–9 one at a time in that cell.
3
isValid check: the digit must not already appear in the same row, same column, or same 3×3 box.
4
If valid, place the digit and recurse to the next empty cell.
5
If recursion returns false, backtrack: reset cell to '.' and try the next digit.
6
If all digits 1–9 fail → return false to the caller (propagate backtrack up).
7
If no empty cell found → board is fully solved, return true.
isValid in Detail
→
Row check: scan all 9 columns in the same row.
↓
Column check: scan all 9 rows in the same column.
⊞
Box check: derive box origin as boxRow = 3*(row/3), boxCol = 3*(col/3), then scan all 9 cells.
Complexity
Time (worst)O(9^m)
SpaceO(m)
m = empty cells≤ 81
