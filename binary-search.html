// Algorithm Details

Classic binary search on a sorted array. Maintain lo and hi pointers. At each step compute mid = ⌊(lo+hi)/2⌋. Compare arr[mid] with target and eliminate half the search space each time.
lo=0, hi=n-1
while lo≤hi: mid=(lo+hi)//2
if arr[mid]==target → return mid
elif arr[mid]<target: lo=mid+1  else: hi=mid-1
